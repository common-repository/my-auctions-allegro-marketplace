== Description ==

= Bridge between two plugins =
- [WC Marketplace] (https://wordpress.org/plugins/dc-woocommerce-multi-vendor)
- [My auctions allegro] (https://wordpress.org/plugins/my-auctions-allegro-free-edition)

= Requirements =

As you know to have online store you need to have WooCommerce plugin, it allows you to sell products online, this is the best option if you have only one offline store and you want to move to sell products online

What about, if you have more offline stores? What about if every store has own products? How to show products assigned to your offline store location?

Helps you plugin [WC Marketplace] (https://wordpress.org/plugins/dc-woocommerce-multi-vendor) it allows to assing product to vendor / your offline store

OK fine, now some features about this plugin, do you sell products on e-commerce site allegro.pl ? Your vendors has own account on this site? This plugin is for you!

Plugin allows you to assing allegro account to WC Markeplace vendor and all imported products from allegro will be automatically assigned to vendor / your offline store

Now imagine that on one site you will have list of your offline stores and below information, list of all available products that this offline store have.

You will be available online and easy to find on the map.

== Installation ==

1. Upload plug-in to wp-content/plugins
2. Enable plug-in in your panel administration
3. This plugin will work only with extension described below, if any of the extension will be not installed, plugin will be not active

== Frequently Asked Questions ==

= How I can use plugin on my site? =

- Install all needed extensions
[WC Marketplace] (https://wordpress.org/plugins/dc-woocommerce-multi-vendor)
[My auctions allegro] (https://wordpress.org/plugins/my-auctions-allegro-free-edition)
[WooCommerce] (https://wordpress.org/plugins/woocommerce)

= How configure plugin to import auctions as product from vendor =

- In My Auctions Allegro -> Plugin Settings you need to edit account and choose vendor from list (only approved vendors will be displayed) and save.

That's it
